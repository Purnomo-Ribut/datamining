Discretization

Discretization atau diskretisasi adalah proses di mana kita dapat mengubah variabel kontinu, model atau fungsi menjadi bentuk diskrit. kita dapat melakukan ini dengan membuat satu set interval yang berdekatan yang melintasi rentang variabel / model / fungsi yang diinginkan.

Pendekatan Discretization

Unsupervised:

a. Equal-Width

pada pendekatan ini data dibagi menjadi k interval dengan ukuran lebar yang sama.

Lebar interval adalah: w = (maks-min) / k

Dan batas intervalnya adalah: min + w, min + 2w, …, min + (k-1) w

b. Equal-Frequency pada pendekatan ini membagi data menjadi kelompok k yang masing-masing kelompok berisi kira-kira jumlah nilai yang sama.

Untuk kedua metode, cara terbaik untuk menentukan k adalah dengan melihat histogram dan mencoba interval atau grup yang berbeda.

Defenisi Entropy-Based Binning

  1. Metode untuk mengelompokkan numeric menjadi categorical

  2. Pengelompokan dengan mencari jumlah yang terbaik

  3. Hasil pemngelompokan terbaik adalah entropi gain yang paling besar

  • Rumus

\[ Entropy (D_1) = - Σ _1^n\ pi \log_2 \ pi \]

Function

Function untuk menghitung banyaknya data setiap kategori

# count every value based the given category
def countEveryCategory(data_column, labels, column, category):
    group = data_column.groupby(category).count()
    amount_of_every_category = []
    for label in labels:
        amount_of_every_category.append(group.loc[label, column])
    return amount_of_every_category

Function menghitung banyaknya data setiap kategori berdasarkan nilai pembagi atau split value yang diberikan

# split the given data based on category and split value 
def split(split_value, data_column, labels, col, category):
    less_group = data_column[data_column[col] < split_value]
    greater_group = data_column[data_column[col] >= split_value]
    
    length_less_group = countEveryCategory(less_group, labels, col, category)
    length_greater_group = countEveryCategory(greater_group, labels, col, category)
    
    return (length_less_group, length_greater_group)

Function menghitung perbadaan gain antara entropi inisial dengan intropinilai pembagi yang diberikan

\[ Gain(Enew) = (Einitial) - (Enew) \]
# count information gain from inisial entropy and new entropy
def count_gain(inisial_entropy, new_entropy):
  return inisial_entropy - new_entropy

Function menghitung entropy

# count entropy
def count_entropy(data_target):
    all_prob = []
    for prob in data_target:
        if (prob/sum(data_target) != 0):
            all_prob.append(prob/sum(data_target) * log2(prob/sum(data_target)))
        else:
            all_prob.append(0)
    return -(sum(all_prob))

Function menghitung info entropy berdasarkan nilai pembagi atau split value yang diberikan

\[ Info_A(D) = \frac {|D_1|}{|D|}\ Entropy(D_1) \frac {|D_2|}{|D|}\ +Entropy(D_2) \]
# count entropy for given a split value
def info(d, data_target):
    temp = []
    for value in d:
        temp.append((sum(value) / data_target.shape[0]) * count_entropy(value))
    return sum(temp)
